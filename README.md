# Smart Cold Storage Management System

## Overview

Smart Cold Storage Management System is a full-stack web application developed to manage warehouse operations, inventory tracking, staff management, and storage monitoring.

## Features

* User Authentication
* Warehouse Registration
* Staff Management
* Product Management
* Inventory Tracking
* Temperature Monitoring
* Complaint Management
* Reports and Alerts
* Admin Dashboard

## Technologies Used

### Frontend

* React.js
* CSS Modules
* Vite

### Backend

* Node.js
* Express.js

### Database

* MongoDB

## Project Structure

* Client (Frontend)
* Server (Backend)
